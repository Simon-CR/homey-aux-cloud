# Asset Images

This directory should contain the following image files:

## App Images (assets/images/)
- `small.png` (250x175px) - App icon for small display
- `large.png` (500x350px) - App icon for medium display
- `xlarge.png` (1000x700px) - App icon for large display

## Driver Images (drivers/aux-ac/assets/images/)
- `small.png` (75x75px) - Device icon for small display
- `large.png` (250x250px) - Device icon for medium display
- `xlarge.png` (500x500px) - Device icon for large display

These images should feature the AUX logo or an air conditioner icon with the AUX blue color (#00A9E0).

For development purposes, you can use the icon.svg as a reference or generate PNG files from it.
